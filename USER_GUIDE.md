# FranceGo — 功能与使用手册

线上地址：**https://infr.europeanaialliance.org**

一个面向法国市场的 **AI 驱动商业拓展 / 市场进入平台**。把法国的企业、招标、新闻、行业统计等公开数据，叠加 AI 分析，帮你发现客户、分销商、合作伙伴、招标机会与决策者，并生成市场进入策略。

> 各模块的数据真实性（哪些是实时真实数据、哪些是占位/示意）见 [DATA_SOURCES.md](DATA_SOURCES.md)。
> 部署与运维见 [DEPLOYMENT.md](DEPLOYMENT.md)。

---

## 1. 快速开始

1. 打开站点 → **注册**（邮箱 + 密码）或**登录**。
2. **语言切换**：右上角下拉 `EN / FR / 中文`，整站界面随之切换；新闻、动态流等的法语内容会**自动翻译**成你选的语言。
3. **暗黑/明亮模式**：右上角太阳/月亮图标切换。
4. 左侧是分组导航：**概览 · 情报 · 商机 · 触达**。
5. 移动端：侧栏收起为汉堡菜单，所有列表都做了移动端卡片布局。

## 3. 模块详解

### 🏠 Dashboard（仪表盘 / 机会雷达）
回答「今天我该做什么来拓展法国业务」。
- **顶部**：机会评分、市场活跃指数、今日新增机会；快捷按钮（找客户 / 找分销商 / 找招标 / 生成报告 / 问 Copilot）。
- **KPI 卡**：今日新机会、匹配招标数、高意向买家、融资/招聘信号等（带 `●live` 圆点的是真实数据）。
- **商业拓展动态流**：实时法国商业新闻（Google News），按信号分类（投资/招标/合作/扩张/风险），可点进原文；标题随界面语言翻译。
- **高意向买家**：正在发布招标的真实采购方（BOAMP），最强购买信号；机构名 + 行业 + 招标标题（翻译）。

### 🌍 Markets（市场情报，模块 1）
- **浏览**：11 个行业卡片（AI、机器人、网络安全、医疗、教育、制造、奢侈品、零售、食品、能源、交通），显示**真实 Eurostat 法国分行业增加值**与同比增长 + 机会/进入难度评分。
- **行业详情页**（点卡片右上「查看详情」）：Eurostat 多年增加值走势图、预测、**该行业真实企业**（可点进公司档案）、行业实时新闻、一键 AI 市场分析。
- **关键词搜索**（页面顶部搜索框）：输入任意关键词（如 fintech、biotech）→ 该领域真实企业 + 实时新闻 + AI 分析；命中内置行业时给出 Eurostat 完整数据入口。

### 🏢 Companies（企业情报，模块 3）
- **搜索**：按公司名 / SIREN / SIRET 实时查法国企业登记库（recherche-entreprises），返回真实身份、营业额、机会评分；展开行可见增值税号、财务年度、高管。
- **完整档案页**（「查看完整档案」）：概览（状态/法律形式/类别/NAF/员工/成立/机构数/增值税号）、自然人高管、**法人股东（可点进其公司）**、多年真实财务柱状图、**BODACC 法律事件**（破产/变更等）、**公司新闻**、一键 AI 商业简报。
- 💎 **增值服务 ·「在法国注册公司」**：页面下方的指引卡片，含 6 步真实注册流程（法律形式→章程→存资本→法定公告→INPI 一站式窗口→SIREN/Kbis→增值税）、预估时长，以及**「通过认证合作伙伴办理」**按钮（链接可配置，见 §6）。

### 🏷️ Brands（品牌情报，模块 4）
- 商标/品牌名检索，返回 风险评分 / 相似度 / 可注册性 + 注册建议（INPI/EUIPO/WIPO 口径）。
- 配置 `EUIPO_*` 密钥后走真实 EUIPO 数据，否则为演示生成。
- 💎 **增值服务 ·「在法国注册商标」**：页面下方的指引卡片，含 6 步真实注册流程（INPI 在先检索→尼斯分类→INPI/EUIPO 提交→审查→2 个月异议期→注册 10 年）、预估时长，以及**「通过认证合作伙伴办理」**按钮（链接可配置，见 §6）。

### 🛡️ Credit（信用情报，模块 12）
- 输入公司名 → 信任总分 + 多维评分：
  - **财务健康**：由 data.gouv.fr 真实财务驱动（界面 `●live`）；
  - **法律风险**：由 **BODACC 真实法律事件**驱动，并列出近期公告；
  - 付款风险 / 供应商可靠性 / 成长：当前为占位评分。

### 📰 News Radar（新闻雷达，模块 15）
- 实时抓取法国商业新闻（Google News），用关键词把每条**分类成信号**（买入/招标/合作/投资/扩张/风险）+ 机会评分。
- 中/英界面**自动翻译标题**（法语原文可一键切回）；可点原文。
- 「加载更多」逐批展开。

### 📋 Opportunities / Tenders（招标情报，模块 5）
- 顶部切换数据源：**BOAMP（法国）** / **TED（欧盟）**，均为实时。
- 关键词搜索招标，显示采购方、截止日、匹配度、中标概率，可点进原公告。
- 「加载更多」。

### 🎯 Discovery（机会发现引擎，模块 2）
- 输入 产品 / 行业 / 目标市场 → 输出 8 类**真实法国企业**：潜在客户、分销商、集成商、伙伴、投资者、加速器、协会、公共采购方，各带评分与理由。
- 「加载更多」展开每类更多企业；公司可点进档案。

### 🔥 Buying Intent（买家意向，模块 9）
- 聚合 **BOAMP 实时招标的采购方**（正在采购 = 最强购买信号）：意向 / 紧迫度 / 销售评分 + 推荐行动。
- 标签与招标标题随界面语言翻译；「加载更多」。

### 📅 Events（事件情报，模块 10）
- 真实法国展会/会议（VivaTech、SIAL、Pollutec、Paris Air Show 等）+ **Google News 实时动态**（最新报道）+ 匹配度/预期线索/商业价值。
- 「加载更多」。

### 🕸️ Network（关系与联系人情报，模块 8/11）
- 搜公司 → **真实高管**（姓名 + 职务，来自 RNE）、法人股东（母公司，可点进）、下属机构数、影响力/购买意向/关系度评分。

### 🤖 Copilot（市场进入 Copilot，模块 13/14）
- **对话**：像顾问一样提问（如「为我的机器人产品在法国找分销商」），由配置的大模型回答（按你界面语言）。
- **多智能体编排**：顶部「AI 智能体团队」（Market/Company/Brand/Tender/Customer/Partner/Contact/Credit 8 个），输入主题点「生成市场进入报告」→ 智能体依次激活 → 生成完整报告，可打印 PDF。

### 📄 Reports（报告中心）
- 8 种模板：公司 / 尽调 / 市场进入 / 行业 / 伙伴 / 招标 / 信用 / 销售机会。
- 选模板 + 输入主题 → AI 生成结构化报告 → 可**打印为 PDF** 或**导出 .md**。

### 💰 Funding（补贴/扶持资金匹配）
- 输入 行业 / 阶段 / 需求 / 地区 → 匹配 France 2030、Bpifrance、CIR/CII、ADEME、区域补贴等并打分，每条带官方链接。

### 📈 Signals（融资动向）
- 列出**刚完成融资的法国公司**（强购买信号），含金额/轮次/意向分，可点进企业详情，一键收藏。也会并入「买家意向」页统一排序。

### 📋 Compliance（落地合规清单）
- 选行业 → 法律形式 / TVA·税务 / 雇佣法 / 行业监管认证 / GDPR 五栏清单 + 各机构**官方链接**；**一键导出 PDF**（中英双语正文）。底部可留资请我们代办公司注册（咨询+落地）。

### ⭐ Watchlist（关注列表 / 轻量 CRM）
- 收藏企业 / 招标 / 机会 → 看板按阶段（潜在→接触→商谈→赢单/丢单）管理，可加备注、改阶段、移除。

### 🛡️ Admin（管理后台，仅管理员可见）
- 用户列表、事件流（注册/升级/退订/留资）、留资线索（可改状态）、KPI，以及 Google Analytics 入口。

### ⚙️ Settings（设置）
- 账户、当前套餐、今日已用搜索数；
- **AI 引擎**卡片：显示当前生效的 provider/模型（绿色=已生效，黄色=mock/未配置）；
- **每日机会邮件**：开关 + 关键词，订阅后每天收到「法国机会雷达」邮件（新招标/融资/新闻）；
- **套餐与升级**：在线支付已开通时显示升级按钮（Stripe 结账），可进计费门户管理/退订。

## 4. 多语言与 AI 翻译
- 界面三语：英语 / 法语 / 中文（next-intl）。
- 数据里的法语内容（新闻标题、招标标题、行业/信号标签）在中/英界面**自动翻译**，专有名词（公司/品牌名）保留原文。
- 翻译由当前 AI 引擎完成；不计入配额。

## 5. AI 引擎（在线大模型）
- 默认本地 Ollama；生产用在线模型（DeepSeek / OpenAI / 通义千问 / Claude）。
- **切换方式**：编辑服务器 `.env` 的 `AI_PROVIDER` + 对应 `*_API_KEY`，重启（详见 [README](README.md) 的「如何切换 / 更新大模型」与 [DEPLOYMENT.md](DEPLOYMENT.md)）。
- 任一模型不可达会自动回退 mock，不影响站点运行。

## 6. 管理员 / 运维速查
- **数据库** PostgreSQL（库 `france_os`）。改用户角色/套餐：
  ```sql
  UPDATE "User" SET role='ADMIN', plan='ENTERPRISE' WHERE email='...';
  ```
- **更新上线**：本地 `git push` 到 main → GitHub Actions 自动部署；或服务器 `npm run deploy`。
- **配置增值服务的合作伙伴链接**：在服务器 `.env` 设
  ```env
  NEXT_PUBLIC_PARTNER_COMPANY_URL="https://合作伙伴-公司注册页"
  NEXT_PUBLIC_PARTNER_BRAND_URL="https://合作伙伴-商标注册页"
  # 也可只设通用的 NEXT_PUBLIC_PARTNER_URL 作为两者的兜底
  ```
  ⚠️ 这是 `NEXT_PUBLIC_` **构建时**变量，改后需重新构建（`npm run deploy`）才生效；未配置时按钮为占位 `#`。
- **进程**：`pm2 status` / `pm2 logs france-os` / `pm2 reload france-os`。
- **数据源/合规**：见 [DATA_SOURCES.md](DATA_SOURCES.md)（含 BODACC 个人数据、Google News 商用等注意事项）。
